# Components

This directory contains reusable React components for the Hynie Nail Room landing page.

## Structure

- `ui/`: Core UI primitives (buttons, inputs, etc.)
- `layout/`: Layout-specific components (header, footer, etc.)
- `sections/`: Large page sections (hero, gallery, etc.)

## Usage

You can import components using the `@/components/` alias:

```tsx
import { Button } from '@/components/ui/button';
```
